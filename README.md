<h1 align="center">lorlid 自用组件库</h1>

## 介绍

这是模仿 [kobalte](https://github.com/kobaltedev/kobalte) 做的一个学习并自用的组件库。

组件采用 [BEM](http://getbem.com/) 命名规范，样式使用 [SCSS](https://sass-lang.com/) 编写。

比如，组件 `FlipGroup` 的样式类名为 `lo-flip-group`，其中的子组件 `FlipGroup.Button` 的样式类名为 `lo-flip-group__button`。

如果是其它情况，会特别标注出来，比如 `FlipGroup.Context` 中的 `item`，会写成 `lo-flip-group__item`。
