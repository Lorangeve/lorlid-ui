## API 参考

### FlipGroup

| 属性       | 类型                             | 说明                                                                                      |
| ---------- | -------------------------------- | ----------------------------------------------------------------------------------------- |
| `children` | `JSX.Element`                    | 组件的子元素，通常包括 `<FlipGroup.Button>` 和 `<FlipGroup.Content>`。                    |
| `shuffle`  | 「可选」`(list: any[]) => any[]` | 一个函数，用于定义如何打乱内容列表的顺序, 不提供的则使用默认 _Fisher-Yates 算法_ 的函数。 |

### FlipGroup.Button

| 属性      | 类型                 | 说明                       |
| --------- | -------------------- | -------------------------- |
| `onClick` | 「可选」`() => void` | 点击按钮时触发的回调函数。 |

## 渲染元素

| 组件                  | 默认渲染元素 |
| --------------------- | ------------ |
| `<FlipGroup>`         | `-`          |
| `<FlipGroup.Button>`  | `<button>`   |
| `<FlipGroup.Content>` | `<ul>`       |
